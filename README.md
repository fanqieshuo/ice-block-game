# 冰块游戏

这是一个简单的HTML5冰块滑行游戏。

## 游戏规则

- 请拖动冰块到灰色空位
- 每个冰块只能移动1次
- 已移动的冰块（紫色）只能退回原位置
- 冰块可以跨过其他冰块或空位移动
- 游戏目标是填满所有非原始位置的空格

## 特性

- 触摸屏支持
- 响应式设计
- 可视化提示可能的移动路径
- 胜利动画效果

## 技术栈

- HTML5
- CSS3
- 原生JavaScript

## 如何开始

只需在浏览器中打开`ice-block-game.html`文件即可开始游戏。
